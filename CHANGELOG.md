# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added

- **module-networks-infrastructure**: Subnets with NAT Gateway, service endpoints, and delegations
- **module-private-endpoins-infrastructure**: Private Endpoints with DNS zone configuration
- **module-storage-infrastructure**: Storage Accounts with blob containers and network rules
- **module-secrets-infrastructure**: Key Vault with secrets management and GitHub sync
- **module-appservices-infrastructure**: App Service Plans with Web Apps and Function Apps
- **module-container-apps-infrastructure**: Container Apps with managed identity and Key Vault integration
- **module-cosmos-infrastructure**: Cosmos DB (SQL API) with geo-replication and diagnostics
- **module-cosmos-postgres-sql-infrastructure**: Cosmos DB for PostgreSQL (Citus) clusters
- **module-sqlserver-infrastructure**: Azure MSSQL Server with geo-replication and failover groups
- **module-events-hubs-infrastructure**: Event Hubs with schema registry and capture
- **module-app-configuration-infrastructure**: App Configuration with feature flags and replicas
- **module-stream-analytics-infrastructure**: Stream Analytics with Event Hub I/O
- **examples/complete/**: Full multi-module deployment example
- Root documentation: README, CONTRIBUTING, LICENSE, CHANGELOG
